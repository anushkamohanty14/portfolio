# Anushka Mohanty — Portfolio

Personal portfolio website built with React and React Router.

**Live site:** https://anushkamohanty14.github.io/portfolio

## Pages

- **About** — bio, research interests, stats, skills, and contact details
- **Projects** — 6 projects with descriptions, tech stack, and GitHub links
- **Experience** — internships, research, and leadership timeline

## Tech Stack

- React 18
- React Router v6 (HashRouter)
- Vite
- Deployed via GitHub Pages + gh-pages

## Run Locally

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run deploy
```
